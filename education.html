Masters of Cyber Security

My advanced studies in Cyber Security provided comprehensive knowledge and hands-on experience in key areas, including:

  • Cyber Security: This paper introduced the key topics in the important field of cybersecurity. It covered various aspects of defensive, offensive, and preventative security. Selective topics from a wide range, such as security vulnerabilities, secure software, intrusion detection and prevention, cryptography, applied cryptography, authentication and access control, information security management, etc., were covered.

  • Information Visualization: This paper aimed to raise awareness of the potential offered by information visualization techniques, develop familiarity with the underlying concepts, and foster the understanding and ability to effectively design and apply information visualizations in a given context.

  • Legal Aspects of Cyber Security: This research seminar paper provided an advanced-level examination of critical issues regarding the legal aspects of Cyber Security. Students were to complete a supervised research project of up to 12,500 words. My supervised research project was based on "illicit activities primarily relating to cybercrime on the Dark Web".

  • Cryptography: This paper aimed to provide a comprehensive understanding of cryptography by exploring its fundamental principles and examining various cryptographic algorithms. It discussed the applications of these algorithms in securing communication, covered the mathematical foundations of cryptography, and addressed key management practices. Additionally, it included practical exercises for designing and implementing cryptographic solutions in real-world scenarios, while also considering emerging topics in the field.

  • Security for Advanced Networks: Networks have evolved considerable in the last decade, leading to new technologies. Changes in the attack surface make network security a permanent concern. Novel approaches to protect them are required, and these approaches were covered in this paper.

  • Malware Analysis and Penetration Testing: This paper went through two important aspects of Cyber Security. It first covered the defensive security techniques of malware analysis and then examined the offensive security techniques of penetration testing.

  • Secure Cloud Application Engineering: Software Engineering on the Cloud is extremely prevalent nowadays. In this paper, we learned how to design, develop, and test cloud-native applications with a special focus on satisfying business requirements.

  • Dissertation: Using machine learning to identify keyboard keystrokes based on sound is a novel approach in cybersecurity research. It contributes to understanding and mitigating acoustic side-channel attacks by analyzing subtle sound variations generated by keystrokes. This method aims to uncover vulnerabilities that could be exploited by malicious actors seeking unauthorized access to sensitive information. By correlating acoustic signals with specific keyboard inputs, we can develop more robust defense mechanisms against these types of attacks.

My studies also covered key frameworks and standards critical to cyber security:

  • NIST Guidelines: We explored the National Institute of Standards and Technology (NIST) guidelines for managing and reducing cybersecurity risks, including the NIST Cybersecurity Framework, which helps organizations identify, protect, detect, respond to, and recover from cyber threats.

  • OWASP: The course included an in-depth look at the Open Web Application Security Project (OWASP), particularly the OWASP Top 10, which highlights the ten most critical web application security risks. This resource is essential for understanding and mitigating common vulnerabilities in web applications.

  • MITRE ATT&CK Framework: We examined the MITRE ATT&CK Framework, a comprehensive knowledge base that details adversary tactics and techniques. This framework aids in understanding, detecting, and defending against cyber threats by providing insights into attacker behavior.

  • ISO 27001/27002: The course also covered ISO 27001 and ISO 27002, international standards for managing sensitive information and implementing best practices for information security controls. ISO 27001 focuses on the requirements for an Information Security Management System (ISMS), while ISO 27002 provides guidelines for selecting and managing security controls based on an organization's risk environment.

This education has equipped me with a robust skill set to tackle complex security challenges in the digital landscape.

Bachelor of Information Technology

My academic journey in IT has equipped me with a strong foundation in both theoretical and practical aspects of the field. All courses include:

  • Fundamentals of Information Technology: Covered basic IT concepts, terminology, and the foundational role of technology in various sectors such as business, healthcare, and education. Topics included computer hardware and software essentials, digital communication technologies, and an introduction to information systems.

  • Programming Principles: Introduced fundamental programming concepts essential for developing software applications. Topics included variables, data types, control structures (loops, conditionals), functions, and basic algorithm design and analysis. Emphasis was placed on problem-solving through structured programming techniques.

  • Introduction to Programming: Provided an introduction to programming languages such as Python, Java, and C#, focusing on syntax, logic, and fundamental programming paradigms (procedural, object-oriented, functional). Gaining hands-on experience writing and debugging simple programs.

  • Introduction to Databases: Introduced principles of database management systems (DBMS), emphasizing database design, data modeling with ER diagrams, SQL querying for data manipulation (SELECT, INSERT, UPDATE, DELETE), and basic database administration concepts.

  • Data Organization: Explored methods for organizing and managing data efficiently, including data storage structures (files, databases), data retrieval techniques (indexing, searching), and basic data manipulation operations (sorting, filtering).

  • Interpersonal Skills: Developed essential interpersonal communication skills critical for teamwork and collaboration in an IT environment. Topics included effective communication strategies, conflict resolution, teamwork dynamics, and professional etiquette.

  • Business Communications: Covered effective communication practices in business settings, including written communication (reports, emails), oral presentations, and interpersonal communication skills necessary for interacting with stakeholders, clients, and team members.

  • Ethics and Professionalism: Explored ethical considerations and professional responsibilities specific to the field of information technology. Topics included ethical decision-making frameworks, privacy concerns, intellectual property rights, and legal issues related to IT.

  • Hardware Fundamentals: Provided foundational knowledge of computer hardware components and architecture. Topics included CPU, memory, storage devices, input/output devices, motherboard architecture, and basic troubleshooting techniques.

  • Internet: Explored the architecture, protocols, and services of the internet. Topics included TCP/IP protocol suite, DNS (Domain Name System), HTTP/HTTPS, web servers, email protocols, and security considerations in internet communications.

  • Mathematics for Information Technology: Covered mathematical concepts and methods relevant to IT applications. Topics included discrete mathematics (sets, relations, functions), algebra (linear equations, matrices), probability and statistics, and their applications in data analysis and decision-making.

  • Systems Overview: Provided an overview of computer systems, including hardware components, operating systems, networks, and their integration to support business and organizational needs. Topics included client-server architecture, distributed systems, and virtualization technologies.

  • Introduction to Systems Analysis and Design: Introduced methodologies for analyzing business requirements and designing information systems solutions. Topics included requirements gathering, feasibility analysis, system design (data flow diagrams, entity-relationship diagrams), and software development life cycle (SDLC) models.

  • Business Applications: Explored the application of IT in business environments to support business processes, decision-making, and competitive advantage. Topics included enterprise resource planning (ERP) systems, customer relationship management (CRM) software, and business intelligence tools for data-driven insights.

  • Networking Principles: Covered computer networking principles and protocols essential for understanding network architecture, communication technologies, and network security. Topics included OSI (Open Systems Interconnection) model, TCP/IP protocol suite, LAN/WAN technologies, routing, switching, and network administration.

  • Operating Systems: Explored the principles and functions of operating systems (OS). Topics included process management (process scheduling, synchronization), memory management (virtual memory, paging), file systems (file organization, access methods), and OS security mechanisms.

  • Problem Solving: Developed skills in algorithmic problem-solving and computational thinking. Emphasized algorithm design techniques (greedy algorithms, dynamic programming, divide and conquer) and data structures (arrays, linked lists, stacks, queues) for efficient problem-solving.

  • Programming Practice: Provided hands-on experience in software development practices and methodologies. Topics included software development life cycle (SDLC), agile methodologies, version control systems (e.g., Git), unit testing, debugging techniques, and code optimization.

  • Database Management Systems: Explored advanced database concepts such as relational database design principles (normalization, indexing), transaction management (ACID properties), concurrency control, database security (authentication, authorization), and data warehousing concepts.

  • Data Analysis: Covered techniques for analyzing and interpreting data using statistical methods (descriptive and inferential statistics), data visualization tools (charts, graphs), and data mining algorithms (clustering, association rule mining) for extracting actionable insights from large datasets.

  • Website Development: Introduced web development technologies and frameworks for building interactive and responsive websites. Topics included front-end technologies (HTML, CSS, JavaScript), back-end scripting languages (PHP, Python, Node.js), database integration, and web application security best practices.

  • Information Gathering: Covered techniques for gathering and analyzing information from various sources to support decision-making and organizational goals. Topics included data collection methods (surveys, interviews, observation), data validation, and data quality assessment.

  • Website Administration: Provided skills in managing and maintaining web servers and applications in a production environment. Topics included web server configuration (Apache, Nginx), domain management, SSL/TLS certificates, web application deployment strategies, and performance optimization.

  • UX Design: Explored principles of user experience (UX) design, usability testing methodologies, user interface (UI) design principles, and human-computer interaction (HCI) concepts. Topics included user-centered design processes, wireframing, prototyping, and accessibility standards.

  • Multimedia Development: Covered multimedia technologies and tools for creating and editing digital media content. Topics included graphics design principles, audio/video editing techniques, multimedia file formats, and integration of multimedia elements into web and mobile applications.

  • Communications for Information Technology: Explored communication technologies, protocols, and strategies used in IT environments. Topics included network communication fundamentals, TCP/IP protocols, network security mechanisms (firewalls, VPN), and wireless communication standards (Wi-Fi, Bluetooth).

  • Networking Practice: Provided practical experience in configuring and troubleshooting computer networks through hands-on labs, simulations, and network design projects. Topics included network configuration (routers, switches), troubleshooting techniques, network monitoring, and performance optimization.
